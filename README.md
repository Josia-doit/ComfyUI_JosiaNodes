# Josia ComfyUI 自定义节点集
一套功能完整、轻量稳定、高度兼容最新版 ComfyUI 的实用节点集。专为解决老旧节点包兼容性差、体积臃肿、运行卡顿问题而设计，把高频刚需功能浓缩为一套轻量化扩展，让 ComfyUI 工作流更简洁、更顺畅。

---

## 📂 完整文件结构
```
ComfyUI_JosiaNodes/
├── __init__.py                    # 节点总注册、统一导出、加载日志
├── node_properties.py # 全节点常量、尺寸预设、描述文案
├── cache_cleanup.py # 显存 / 内存 / 系统缓存清理节点
├── seed.py # 随机种子管理节点
├── flow_valve.py # 5 通道流量阀门节点（Python 逻辑）
├── encoder.py # 文本 + 多图参考编码节点
├── image_comparer.py # 双图对比预览节点
├── image_scaling.py # 多功能图像缩放裁切节点
├── group_controller.py # 分组控制节点（多组 + 单组）
│
└── web/
    └── js/
        ├── flow_valve.js # 流量阀门前端美化
        ├── seed.js # 随机种子前端按钮、快捷操作
        ├── encoder.js # 文本编码节点默认尺寸配置
        ├── image_comparer.js # 图像对比滑动 / 按住对比交互
        └── group_controller.js # 分组控制前端 UI、开关、导航逻辑
```

---

## 📋 节点列表与功能说明

### 1. Josia流量阀门（JosiaFlowValve）
- **分类**：Josia 专用节点/流量控制
- **核心功能**：5 路独立通道开关，自由控制数据透传或截断
- **输入**：通道1~5开关、输入1~5（任意类型，可选）
- **输出**：输出1~5（按开关状态返回对应数据）
- **特点**：全类型兼容、可视化开关状态、前端自动美化、容错性极强

---

### 2. Josia随机种子（JosiaSeed）
- **分类**：Josia
- **核心功能**：专业级随机种子管理，支持自动/递增/递减/固定模式
- **种子规则**：
  - `-1`：每次运行自动随机新种子
  - `-2`：基于上一次种子自动 +1
  - `-3`：基于上一次种子自动 -1
  - 其他数值：固定种子，稳定复现结果
- **前端功能**：
  - 🎲 每次随机
  - 🎲 新固定随机
  - ♻️ 使用上一次种子（快捷按钮）
- **已知问题**：自动回填上一次种子功能暂未生效，欢迎社区协助修复

---

### 3. Josia文本编码（JosiaEncoder）
- **分类**：JosiaNodes
- **核心能力**：文生图+图生图一体化编码，支持最多5张参考图融合，新增Latent输出（适配Flux2 Klein等模型）
- **核心特性**：
  - 图像参考开关：一键切换文生图/图生图模式
  - 负向提示词开关：关闭后自动归零，避免负面干扰
  - 多图平均融合，参考强度精细调节
  - Latent输出：✅主图潜空间Latent / ❌仅主图尺寸Latent
- **优势**：原生逻辑、高度兼容、无额外依赖、前端自动适配

---

### 4. Josia图像对比（JosiaImageComparer）
- **分类**：Josia
- **核心功能**：双图实时对比预览，支持两种交互模式
- **模式**：
  - Slide 模式：鼠标滑动显示竖直分割线
  - Click 模式：按住显示 B 图，松开显示 A 图
- **特点**：继承官方预览逻辑、自动保存、前端渲染流畅、支持拉大节点舒适对比

---

### 5. Josia图像缩放（JosiaImageScaling）
- **分类**：Josia
- **核心功能**：全能图像缩放裁切，支持 4 大比例 + 3 种自定义模式
- **预设比例**：
  - 1:1 正方形
  - 2:3 / 3:2 摄影比例
  - 3:4 / 4:3 短视频比例
  - 16:9 / 9:16 全平台视频
- **缩放模式**：
  - 📏 边长缩放（按最长/最短边）
  - 🖼️ 像素缩放（按百万像素自动算尺寸）
  - ✏️ 手动宽高（支持一键切换宽高）
- **高级特性**：
  - 锁定 2/4/8/16/32/64/128 倍数
  - 中心/上下左右裁剪 + 拉伸缩放
  - 5 种缩放算法（nearest / bilinear / area / bicubic / lanczos）
  - 渐进式多步缩放，低像素更清晰
  - 自动分辨率保护，上限 400 万像素
- **输入输出**：图像、遮罩、Latent 全支持

---

### 6. Josia分组控制（JosiaGroupControllerM / JosiaGroupControllerS）
- **分类**：Josia Nodes/group_control
- **多组控制**：自动扫描所有编组，一键全部跳过/启用，点击组名快速定位
- **单组控制**：下拉选择编组，单个开关精准控制启用/跳过
- **状态显示**：
  - 绿色：已启用
  - 红色：已跳过
  - 橙色：部分跳过
- **特点**：纯前端交互、不占算力、支持右键菜单、状态随工作流保存

---

### 7. Josia缓存清理（JosiaCacheCleanup）
- **分类**：Josia
- **核心功能**：轻量安全释放显存/内存/系统缓存，不卸载模型
- **清理项**：
  - 显存缓存（全平台）
  - Windows 系统文件缓存
  - Windows 非系统进程空闲内存
- **特点**：
  - 任意数据透传，串联不中断工作流
  - 自动识别系统，非 Windows 自动安全降级
  - 实时日志输出，耗时统计

## 🛠️ 安装方法
1. 下载整个 `ComfyUI_JosiaNodes` 文件夹
2. 放入 `ComfyUI/custom_nodes/` 目录下
3. 确保 `web/js` 文件夹完整，不要移动或删除文件
4. 重启 ComfyUI
5. 看到控制台输出 `[JosiaNodes] ✅ 全部节点加载完成` 即成功

## 🎯 设计理念
- **轻量不臃肿**：拒绝冗余，只保留真正高频刚需功能
- **新版高兼容**：专为最新 ComfyUI 开发，不依赖老旧库
- **低占用流畅**：自动 GC、显存优化、前端轻量化
- **中文友好**：全中文界面、符号可视化、逻辑直白
- **稳定安全**：不删文件、不杀进程、不破坏工作流

## ⚠️ 重要说明
1. 随机种子节点「♻️ 使用上一次种子」功能目前失效，欢迎社区提交 PR 修复
2. 图像缩放节点内置分辨率上限保护，避免显存溢出
3. 缓存清理节点在 Windows 上优化更明显，其他系统仅清理显存
4. 分组控制、图像对比、流量阀门、种子节点均依赖前端 JS 文件，不可删除
